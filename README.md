# Buscador de bebidas

Programa el cual busca una bebida por ingrediente

## Tecnologías usadas

- __React__
- __TailwindCSS__
- __TypeScript__

## Guía de instalación

```
git clone https://github.com/aidancom/buscador-bebidas.git
cd buscador-bebidas
npm i
npm run dev
```

__Importante__: Asegurate de tener Node.js instalado
